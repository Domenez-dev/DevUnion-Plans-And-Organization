## Main Channels And Threads
#### **📌 1. Information** (Read-Only)

- **#📢・announcements** → Important updates & events
- **#📜・rules** → Server rules & guidelines
- **#👋・welcome** → Auto-welcome new members
- **#📝・roles** → Reaction-based roles (tech stack, interests, etc.)

---

### **💬 2. General Discussions**

- **#💬・general** → Main chat for all topics
- **#🤝・introductions** → Members introduce themselves
- **#💡・suggestions** → Feedback & improvements

---

### **💻 3. IT & Tech Discussions**

- **#🐍・python** → Python discussions
- **#⚛️・web-dev** → Frontend, backend, and full-stack topics
- **#🛠️・devops** → Cloud, servers, deployment
- **#🕵️・cybersecurity** → Ethical hacking, security news
- **#🤖・ai-ml** → AI, ML, and data science discussions
- **#📰・tech-news** → Share IT-related news

---

### **🛠️ 4. Help & Support**

- **#❓・help** → Members can ask coding questions
- **#💡・resources** → Free books, tools, and courses
- **#📝・study-group** → Learning & group projects

---

### **🚀 5. Community Activities**

- **#🎯・challenges** → Coding challenges, hackathons
- **#🗣️・events** → Webinars, meetups, discussions
- **#🎤・ama** → Ask-Me-Anything with experts

---

### **🎨 6. Fun & Off-Topic**

- **#🎮・Random Chat** → off topic chat
- **#🎮・gaming** → Casual gaming discussions
- **#📺・memes** → IT-related memes
- **#📚・Books** → For book lovers
- **#📷・showcase** → Share projects, designs

---

### **🤝 7. Collaborators Space

- **🤝・Discussion** → For Collaborators to Chat
- **🐈️・GitHub Logs** → To get and notified about the GitHub repository
- **🔊・Voice-Room** → Voice room For collaborators

---

### **🔊 8. Voice Channels**

- **🔊・general** → Open voice chat
- **🔊・coding-room** → Study sessions & pair programming
- **🔊・events** → Special discussions or podcasts


---

### **👑 9. Private & Staff** (Only for Admins/Mods)

- **#📌・staff-announcements** → Internal updates
- **#🚨・mod-chat** → Team discussions
- **#🛠️・bot-setup** → Logs, bot commands
